# Reviewr
## Overview
An interactive React app to organize, rate, and review digital media like movies, music, and podcasts.

## Goals
I want to get more practice with database interaction and frontend development. I would like for this project to also lean into the internet a little bit, because I want there to be a feed page, where you and your friends can all get together and rate things or post about media you have consumed recently.

## Use of AI
I will be using AI like `Claude.ai` or `Copilot` to debug or aide me in creating some aspects of this application.
